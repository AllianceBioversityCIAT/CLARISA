Information on Dashboards and Services

CLARISA helps collect and transform the raw CGIAR research and activities data into standardized and aggregated information. By providing control lists with codes (i.e., unique identifiers), data on common indicators can be gathered from the systems adopting these codes and results compiled at a system-wide level, including through dashboards.

CLARISA's architectural style enables computer systems to work together over the Internet. In programming terms, CLARISA is based on a REST (Representational State Transfer) - API (Application Programming Interface). Thanks to its architecture, CLARISA unifies the language across different platforms, overcoming the common challenge of non-standardized information.

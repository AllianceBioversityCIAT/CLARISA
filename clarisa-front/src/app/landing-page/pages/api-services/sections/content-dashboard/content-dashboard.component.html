CLARISA Services

CLARISA's architectural style enables computer systems to work together over the Internet. In programming terms, CLARISA is based on a REST (Representational State Transfer) - API (Application Programming Interface).

Thanks to its architecture, CLARISA unifies the language across different platforms, overcoming the common challenge of non-standardized information.

CLARISA uses control lists of standardized key terms, such as countries and regions, acronyms, institutions, budget types, Sustainable Development Goals (SDGs), and CGIAR technical language to enable platforms to speak the same language when exchanging data and information.

An overview of CLARISA's services is offered below:

  • Annual Report Control Lists
  • One CGIAR Control Lists
  • Additional Services
  • One CGIAR Operation
